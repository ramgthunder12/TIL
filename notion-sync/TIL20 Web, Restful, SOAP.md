**웹 서비스** : 네트워크 상에 분산된 자원들을 이기종 간에 서로 연동하여 자원을 공유 하기위한 추상적인 `**서비스의 형태**`

**SOAP 기반(방식) 웹 서비스 : 자원을 공유할 때 메시지를 주고 받는 방식**

**RESTful 기반(방식) 웹 서비스 : 자원을 공유할 때 자원을 주고 받는 방식(자원 중심으로 표현하고, 자원 중심으로 전달하고, 자원 중심으로 접근하는 방식)**
