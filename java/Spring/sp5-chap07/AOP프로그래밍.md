#프록시와 AOP
##AOP 란
Aspect Orientide Programming
관점 지향 프로그래밍
목적에 따라 프로그램을 나누는 방식

##프록시 란
목적을 공통부분과 주제 부분으로 나눠서
공통되는 부분을 재사용하는것
공통 로직을 따로 정의해 재사용하는것

#스프링 AOP구현 방법
## 구현 방법
###pointcut 정의
	어떤 집합(구역)에서 공통기능을 실행 시킬 것인가 정의
###advice 정의
	메인 기능을 실행시킬때 공통기능은 언제 실행 시킬것인가 정의, 단 spring은 around만 가능
	advice 종류
	after : 메인 기능 전에 공통기능 호출
	befor : 메인 기능 후에 공통기능 호출
	around : 메인 기능 전, 후에 둘다 공통기능 호출
	exception : 메인 기능에 예외 발생 시 공통기능 호출
	final : 메인 기능에서 final메소드 호출 시 공통기능 호출
